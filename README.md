# Vehicle-Information-Extractor
Script to pull vehicle information from Liferadt and TLOxp PDF into CSV files.

Usage: Commmand line utility to run in the same folder you have several TLOxp or Liferaft PDFs. It will loop through all PDFs in the folder and create a CSV file with the same name. Example: "information.pdf" will become "information.pdf.csv".

Known Issues:
-  It will not process a TLOxp Person Comprehensive report.
-  Registered Owner information is sometimes wonky.
